Sobre o InboxFlox

O InboxFlox é uma ferramenta demonstrativa para acelerar a triagem de e-mails: você fornece o conteúdo (texto ou arquivo) e recebe instantaneamente uma classificação e uma resposta de rascunho. Focado em mostrar como integrar modelo generativo + parsing robusto + fallback local.

Objetivo

Diminuir o tempo gasto em leitura inicial e rascunho de respostas operacionais.

Pipeline

Upload/Texto → Pré-processa → Chamada Gemini → Extração segura de JSON → Exibição + Copiar.

Fallback

Se a API falha, regras simples mantêm a experiência fluida.

Detalhes Técnicos

  • Flask + fetch API (sem framework front pesado).
  • Parsing resiliente de JSON da resposta (remoção de trechos extras / correção de aspas).
  • Extração de texto de PDF com PyMuPDF.
  • Fallback por regras de palavras-chave.

Limitações

  • Classificação binária simplificada.
  • Respostas genéricas — ideal refinar antes de enviar.
  • Sem armazenamento ou histórico (foco em demonstração).

Possíveis Evoluções

  • Adicionar categorias extras (Follow-up / Spam / Ação urgente).
  • Autenticação + histórico de análises.
  • Exportar lote de resultados em CSV.
  • Dark mode e tema custom.
